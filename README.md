# 🌐 실시간 웹 캐치마인드 배포 가이드 (GitHub & Render 이용)

Glitch 서비스가 종료됨에 따라, 현재 가장 안정적이고 대중적인 무료 클라우드 배포 플랫폼인 **Render(렌더)**를 이용한 배포 방법을 안내해 드립니다.

깃허브(GitHub)와 렌더(Render)를 사용하지만, **복잡한 코딩 명령어(터미널) 없이 인터넷 브라우저 마우스 클릭만으로 배포할 수 있는 초보자용 가이드**입니다.

---

## 🚀 5분 만에 내 인터넷 게임 사이트 만들기

### 1단계: 깃허브(GitHub)에 코드 올리기 (저장소 만들기)
렌더에 웹사이트를 띄우려면 먼저 코드가 깃허브에 올라가 있어야 합니다. (구글 드라이브에 파일을 업로드하는 것과 비슷합니다.)

1. [깃허브(GitHub) 홈페이지](https://github.com)에 접속하여 회원가입(Sign up) 및 로그인을 합니다.
2. 메인 화면 좌측 상단 또는 우측 상단의 **[New]** (또는 **[Create repository]**) 버튼을 클릭하여 새 저장소를 만듭니다.
   * **Repository name** (저장소 이름): `catchmind-web` (원하는 이름 입력)
   * **Public / Private**: `Public` 선택 (무료 배포를 위해 필수)
   * 다른 설정은 건드리지 않고 최하단의 **[Create repository]** 버튼을 클릭합니다.
3. 생성된 화면 중간에 있는 **"uploading an existing file"** 파란색 링크를 클릭합니다.
4. 내 컴퓨터의 `catchmind-web` 폴더 안에 있는 다음 파일들을 브라우저 화면으로 드래그 앤 드롭(마우스로 끌어서 놓기)하여 업로드합니다:
   * `package.json`
   * `server.js`
   * `public` 폴더 전체 (안의 `index.html`, `style.css`, `client.js` 포함)
5. 업로드가 완료되면 화면 하단의 초록색 **[Commit changes]** 버튼을 클릭하여 저장합니다.

---

### 2단계: 렌더(Render)에 연결하여 배포하기
이제 깃허브에 올린 설계도를 바탕으로 실제 24시간 가동되는 서버를 띄울 차례입니다.

1. [렌더(Render) 홈페이지](https://render.com)에 접속합니다.
2. 우측 상단의 **[Get Started]** 또는 **[Sign Up]**을 눌러, **[GitHub] 아이콘을 클릭하여 깃허브 계정으로 간편 가입 및 로그인**합니다.
3. 로그인 후 보이는 대시보드 화면에서 우측 상단의 **[New +]** -> **[Web Service]**를 클릭합니다.
4. **"Build and deploy from a Git repository"** 메뉴를 선택하고 Next를 누릅니다.
5. 깃허브 연동을 허용하면 방금 생성한 `catchmind-web` 저장소가 목록에 보입니다. 그 옆의 **[Connect]** 버튼을 누릅니다.
6. 설정 정보 입력 창이 나옵니다. 아래 항목들만 확인 및 수정해 줍니다:
   * **Name**: 사이트 식별 이름 (예: `our-class-catchmind` - 영어 소문자 및 하이픈만 가능)
   * **Language**: `Node` (기본값)
   * **Branch**: `main` (기본값)
   * **Region**: `Singapore` 또는 `Oregon` (기본값)
   * **Build Command**: `npm install` (기본값)
   * **Start Command**: `npm start`
7. 하단에서 요금제 플랜 중 **[Free]** (무료)가 선택되어 있는지 확인한 후, 최하단의 **[Create Web Service]** 버튼을 누릅니다.

---

### 3단계: 나만의 인터넷 게임 주소 발급 완료!
1. 버튼을 누르면 검은색 화면(로그 창)에 서버가 빌드되는 진행 상황이 표시됩니다. 약 1~2분 정도 소요됩니다.
2. 화면 좌측 상단에 표시되는 파란색 링크(예: `https://our-class-catchmind.onrender.com`)가 **선생님의 고유한 인터넷 주소**입니다.
3. 로그 창에 `CatchMind web-ready server running on port...` 또는 관련 완료 메시지가 뜨고 상태가 **[Live]**로 바뀌면 주소로 접속이 가능해집니다.
4. **이 주소를 학생들에게 보내주시면 언제 어디서든 접속하여 조별 플레이를 즐길 수 있습니다!**

---

## 💡 유용한 팁
* **첫 접속 대기 시간 (Cold Start)**: 무료 요금제의 특성상 약 15분간 접속자가 없으면 서버가 자동으로 정지(잠자기) 상태가 됩니다. 이 상태에서 누군가 다시 주소로 접속하면 깨어나는 데 **약 50초~1분** 정도 시간이 걸립니다.
  * *따라서 수업 시작 5분 전에 선생님이 먼저 사이트 링크에 접속하여 사이트가 완전히 켜진 것을 확인해 두시면 수업 시간에 학생들이 렉 없이 바로 들어올 수 있습니다.*
